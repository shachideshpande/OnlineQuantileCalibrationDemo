The code for each experiment is contained within the examples folder.
The code is contained within these files
1) calibration-fish-toxicity.py
2) calibration-facebook-comment-volume.py
3) calibration-energy-efficiency.py
4) calibration-aquatic-toxicity.py

Each experiment can be run with the command 

python <experiment source code file name>.

The examples folder also contains csv files of the required UCI datasets as cited in the paper. Please make sure that the directory paths are set appropriately in the python code before running the experiments.
